# Band Name Generator 

A simple and fun Python script that helps you come up with a creative name for your next musical project. It uses a classic naming convention: combining the name of your hometown with the name of a pet.

## How It Works

The script follows a linear flow to capture user input and generate a result:

+ Greets the user.

+ Asks for the city where the user grew up.

+ for the name of a pet.

+ Combines these two strings to display a suggested band name.
